폼 관련 태그

form 태그는 html에서 사용자가 입력할 수 있는 양식을 제공하는 태그로
form 태그 내의 input 태그들을 통해 사용자가 입력한 정보를 서버로 넘기는 역할을 함
* action 속성: 폼에 입력된 값들을 전송받을 서버의 클래스 명 입력
* method 속성: get/post 방식으로 전송 방식 지정(디폴트: get 방식)
