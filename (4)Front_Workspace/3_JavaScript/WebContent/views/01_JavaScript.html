자바스크립트 개요

스크립트 언어

기본 프로그램의 동작을 사용자의 요구에 맞게 수행되도록 해주는 용도로 사용
매우 빠르게 배우고 짧은 소스코드 파일로 상호작용 하도록 고안됨

자바스크립트

웹 브라우저에서 많이 사용하는 인터프리터 방식의 객체지향 프로그래밍 언어

자바스크립트 사용법

1. 태그에 직접 간단한 소스 코드를 작성해서 실행하는 방법(inline)
2. Script 태그 안에 작성하여 실행하는 방법(internal)
3. 외부에 별도 js 파일을 작성하여 가져다 사용하는 방법(external)

태그에 직접 자바스크립트 코드를 작성하여 실행

Script 태그 안에 함수 단위로 소스 작성하여 실행

별도의 js 파일로 실행
