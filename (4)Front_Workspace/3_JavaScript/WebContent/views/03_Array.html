배열

자바스크립트에서는 자료형 지정이 없기 때문에 한 배열 안에 모든 자료형을 보관할 수 있다.

배열 선언 및 초기화

배열 선언 시 크기를 지정하지 않고 선언할 수도 있고 트기를 정하고 선언할 수도 있음

배열 메소드

array도 하나의 객체이기 때문에 배열에서 활요할 수 있는 메소드가 있음

indexOf()

배열에서 요소가 위치한 인덱스 리턴

concat()

여러 개의 배열을 결합하는 용도

join()

배열을 결합하여 문자열로 만들어줌

reverse()

배열의 순서를 뒤집음
원본 배열에 영향 미침

sort()

배열을 내림차순 혹은 오름차순으로 정렬
원본 배열에 영향 미침

push()와 pop()

push: 맨 뒤에 요소 추가
pop: 마지막 요소 제거

shift()와 unshift()

shift: 배열의 맨 앞의 요소를 뽑아오는 (출력 가능) 동시에 제거
unshift: 배열 맨 앞에 매개변수로 입력한 요소 초가

slice()와 splice()

slice(): 배열의 요소 뽑아내기 (원본에 영향 x)
splice(인덱스): 배열의 index부터 끝까지 요소를 제거(원본에 영향 O)
splice(인덱스1, 인덱스2, 요소): 배열의 인덱스1 <= index < 인덱스2에 위치한 요소를 제거하고 지정한 요소 삽입 (원본에 영향 O)

toString()

배열을 문자열로 반환
join()과 toString() 모두 배열을 문자열로 반환한다
다만 join()은 매개변수에 넣은 값을 구분자로 반환(매개변수 없으면 ,로 구분)
